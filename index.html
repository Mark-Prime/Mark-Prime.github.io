Ryukbot

Ryukbot is a python program used to automate the recording process of video clips gathered within video games developed with the Source Engine. It uses features built into the engine that allow it to convert .dem files (game replays) into mp4 footage. The process is frame perfect and automated, simply tell it when something happened and it will record the area around it at any FPS (frames per second) you tell it to. This cuts the time it takes to record footage dramatically for people who record often and makes it possible for people who didn't have systems strong enough previously.

Ryukbot also has it's own programming language called RBC for modding purposes. While basic the language allows you to add a lot of customizing to the video recording process.
